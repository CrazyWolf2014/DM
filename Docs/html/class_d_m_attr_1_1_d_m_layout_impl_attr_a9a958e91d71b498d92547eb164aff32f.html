wchar_t* DMAttr::DMLayoutImplAttr::POS_pos
static

指定位置,最常用的为指定四个坐标,示例：

备注
示例1:pos="10,11,101,102",四个点分别代表了左:10,上:11,右:101,下:102
示例2:pos="10,11,-12,-13",四个点分别代表了左:10,上:11,右:父窗口右边左移12,下:父窗口下边上移13—>'-'：定义窗口的计算以相反坐标系
示例3:pos="10,11,@100,@101",四个点分别代表了左:10,上:11,宽:100,高:101—>'@'：定义窗口的size，只在后两个值中有效
示例4:pos="10,11,|-1,|2",四个点分别代表了左:10,上:11,右:父窗口中心点左移1(-1),下：父窗口中心点下移2–>'|':参考父窗口中心
示例5:pos="[10,[11,@100,@101",四个点分别代表了左:距前一个兄弟窗口自己近的边右移(10)，上:距前一个兄弟窗口自己近的边下移(10)，宽:100,高:101—>'['参考前一个兄弟窗口与自己近的边（没有兄弟窗口就参考父窗口的可用布局空间）
示例6:pos="0,1,]-10,@100",四个点分别代表了左:0，上:1，右:距下一个兄弟窗口左移10(-10),高:100—>']'参考下一个兄弟窗口与自己近的边（没有兄弟窗口就参考父窗口的可用布局空间）
–>'{'参考下一个兄弟窗口与自己近的边，与示例5类型，–>'}'参考下一个兄弟窗口与自己远的边,与示例6类似
示例7:pos="%10,%20,%30,%40",四个点分别代表了左:父窗口布局空间的%10，上：父窗口布局空间的%20，右：父窗口布局空间的%30，下：父窗口布局空间的%40—>''采用在父窗口的百分比定义坐标
