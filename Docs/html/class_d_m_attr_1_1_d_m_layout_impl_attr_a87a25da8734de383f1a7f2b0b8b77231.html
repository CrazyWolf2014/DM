wchar_t* DMAttr::DMLayoutImplAttr::SIZE_possize
static

指定位置,在pos坐标小于四时，结合pos使用,示例:

备注
示例1:pos="%10,%11" possize="100,101",pos记录了左上角坐标,而possize记录了宽100，高101
示例2:pos="%10,%20" possize="-1,-1"，pos不为空,同时possize大小为-1，-1,（-1)表示根据内容布局，使用DV_GetDesiredSize，possize前一值为横向，后一值为竖直
示例3:pos="%10,%20" possize="-1,100"，pos不为空,同时possize大小为-1，100,（-1)表示根据内容布局,高度100,使用DV_GetDesiredSize，possize前一值为横向，后一值为竖直
示例4:pos="" possize="-1,-1",pos为空,possize大小为-1，-1，填满父窗口
示例4:pos="" possize="x,x",pos为空,x不全为-1，-1，使用自动排序,前-1表示填满横向父窗口，后-1表示填满坚向父窗口
