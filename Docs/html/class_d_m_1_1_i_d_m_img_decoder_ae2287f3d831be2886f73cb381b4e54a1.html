virtual DMCode DM::IDMImgDecoder::GetTotalLoopCount ( UINT &  ulCount)
pure virtual

返回图片解码后的多帧循环次数

参数
[out]ulCount图片解码后的多帧循环次数(-1表示永远循环)
备注
按GIF的逻辑,三种状态，1.没有循环信息（1次） 2.有循环信息（次数）3.有循环信息（0次表示永远循环）
返回
DM_ECODE_OK为成功
